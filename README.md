- Matching integrated terminal palettes
- OLED-focused variant with true-black backgrounds
- Designed for long coding sessions with comfortable contrast

# Installation

## Install From GitHub

If the theme is not available in the Zed marketplace yet, install it directly from this repository:

1. Download or clone this repository from GitHub.
2. Open Zed.
3. Run `zed: install dev extension`.
4. Select the downloaded repository folder.
5. Enable the theme in Zed's theme picker.
6. Choose either **Passione Dark** or **Passione OLED**.

## Notes

- This extension is not assumed to be published on the Zed marketplace yet.
- Installing from GitHub is the recommended path until publication.
- After installation, you can switch between the two variants from Zed's theme picker.

# Theme Variants

- `Passione Dark`: balanced dark surfaces with editor and terminal matched closely together
- `Passione OLED`: near-black OLED variant with fully blended editor and terminal surfaces

# Quick Summary

- `Passione Dark` keeps the family identity with a softer, more neutral dark surface.
- `Passione OLED` pushes the same design language into near-black territory for OLED displays.
- Both variants share the same syntax palette and integrated terminal styling.
