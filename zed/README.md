# Passion for Zed

Passion ships as a Zed extension with two variants:

- `Passione Dark`
- `Passione OLED`

## Download

Download the Zed package from the repo:

[Download passione-zed-1.2.0.zip](../dist/passione-zed-1.2.0.zip)

## Install

1. Download `passione-zed-1.2.0.zip`.
2. Extract the zip.
3. Open Zed.
4. Run `zed: install dev extension`.
5. Select the extracted `passione-zed-1.2.0` folder.
6. Choose `Passione Dark` or `Passione OLED` in the theme picker.
