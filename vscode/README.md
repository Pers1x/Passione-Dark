# Passion for VS Code

Passion ships as a VS Code color theme extension with two variants:

- `Passione Dark`
- `Passione OLED`

## Download

Download the VS Code package from the repo:

[Download passione-vscode-1.2.0.vsix](../dist/passione-vscode-1.2.0.vsix)

## Install

1. Download `passione-vscode-1.2.0.vsix`.
2. Open VS Code.
3. Run `Extensions: Install from VSIX...`.
4. Select the downloaded `.vsix` file.
5. Run `Preferences: Color Theme`.
6. Choose `Passione Dark` or `Passione OLED`.
